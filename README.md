# Projet

Weather-APP est une application météo qui permet à un utilisateur d'obtenir, en fonction de la ville choisie, la météo actuelle :

- Température resssentie
- Température réelle
- Humidité
- Vitesse du vente
- Couverture nuageuse

## API

Weather-APP utilise l'API OpenWeatherMap.
